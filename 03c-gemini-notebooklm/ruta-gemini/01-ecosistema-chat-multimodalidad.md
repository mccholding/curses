# Módulos 0–1 · Ecosistema, Chat y multimodalidad

**Duración:** 4 horas.

## Objetivos

- Comprender las piezas principales del ecosistema Google AI.
- Elegir entre conversación, investigación, creación y acción programada.
- Trabajar con texto, imágenes, audio, vídeo y archivos.
- Verificar resultados multimodales antes de utilizarlos.

## Mapa de decisión

| Necesidad | Capacidad recomendada |
|---|---|
| consulta o borrador rápido | Chat |
| información actual con investigación extensa | Deep Research |
| documento, presentación, código o prototipo | Canvas |
| comportamiento especializado recurrente | Gem |
| rutina diaria, semanal o mensual | acción programada |
| análisis profundo de un corpus seleccionado | NotebookLM |

## Multimodalidad

La práctica separa cuatro niveles:

1. **Observación:** qué aparece explícitamente.
2. **Extracción:** datos, texto, objetos o estructura.
3. **Interpretación:** qué puede significar.
4. **Inferencia:** conclusiones que necesitan validación.

## Prácticas

1. Comparar el mismo encargo por texto, voz, imagen y archivo.
2. Analizar una imagen sin mezclar observación e inferencia.
3. Extraer información de un documento y contrastarla con la fuente.
4. Identificar qué datos no deberían cargarse en una cuenta no autorizada.

## Prueba adversarial

Entregar un archivo incompleto, una imagen ambigua o instrucciones contradictorias. El sistema debe declarar límites, pedir información y evitar conclusiones falsas.

## Entregable

Un flujo multimodal documentado con:

- objetivo;
- entradas;
- capacidad elegida;
- criterios de verificación;
- riesgos;
- resultado corregido.
