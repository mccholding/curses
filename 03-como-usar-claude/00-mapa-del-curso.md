# Mapa del curso · Claude aplicado

## Propósito

Aplicar el mapa del Curso 02 en cuatro capacidades diferenciales de Claude: Artifacts, Projects, conectores y Cowork.

## Ruta profesional

```text
PRODUCIR
Artifacts y entregables reutilizables
        ↓
ORGANIZAR
Projects, fuentes, instrucciones y continuidad
        ↓
CONECTAR
Herramientas reales con permisos mínimos
        ↓
DELEGAR
Cowork, carpetas, revisión y sistema documentado
```

## Cuatro bloques

| Bloque | Tema | Duración | Producto del alumno |
|---:|---|---:|---|
| 1 | Artifacts y entregables | 2 h | Artifact funcional |
| 2 | Projects y conocimiento | 2 h | Project configurado |
| 3 | Conectores y permisos | 2 h | flujo conectado de bajo riesgo |
| 4 | Cowork y proyecto final | 2 h | encargo completo y manual operativo |

**Duración total:** 8 horas.

## Regla de no repetición

El instructor asume que el alumno ya comprende:

- contexto y fuentes;
- RAG;
- MCP;
- agentes y subagentes;
- loops y condiciones de parada;
- tareas programadas;
- permisos y control humano.

Las explicaciones introductorias de los archivos existentes se utilizan solo como recapitulación breve.

## Especialización Claude Code

### Nivelación opcional

- entorno y permisos;
- `CLAUDE.md` y contexto durable.

### Núcleo avanzado

| Bloque | Enfoque |
|---|---|
| Personalización | settings, Skills y hooks mantenibles |
| Conexión | MCP real y herramientas del equipo |
| Orquestación | subagentes, agent teams y worktrees |
| Distribución | plugins y configuraciones compartidas |
| Automatización | headless, CI/CD y Agent SDK |
| Gobierno técnico | hooks empresariales, seguridad y auditoría |
| Flujo profesional | Git, revisión y entrega |
| Operación | troubleshooting y runbook interno |

## Formatos

### Profesionales no técnicos

Dictar únicamente la Ruta profesional de 8 horas.

### Equipos técnicos

Usar una nivelación breve y continuar con Claude Code.

### Programa mixto

Todos desarrollan el mismo proceso empresarial:

- profesionales diseñan el Artifact, el Project, permisos y operación;
- técnicos implementan Skills, MCP, hooks, automatización y controles.

## Criterios de salida

La persona termina la Ruta profesional cuando puede:

- crear un Artifact útil;
- mantener un Project limpio;
- conectar una fuente con permiso mínimo;
- delegar un encargo completo en Cowork;
- revisar y documentar el sistema.

La persona termina Claude Code cuando puede configurar, automatizar, compartir y operar un flujo técnico con evidencia y capacidad de reversión.
