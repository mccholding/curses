# Ruta A · Claude para profesionales

## Propósito

Dominar Claude como un sistema de trabajo, no únicamente como un chat. La ruta lleva al participante desde tareas individuales hasta proyectos conectados, delegación y adopción responsable.

## Audiencia

- Profesionales de operaciones y administración.
- Consultores y prestadores de servicios.
- Educadores y capacitadores.
- Equipos de marketing y contenido.
- Analistas y líderes de equipo.
- Emprendedores y dueños de PyME.

No requiere programar ni utilizar terminal.

## Prerrequisitos

- Haber usado Claude, ChatGPT o Gemini.
- Conocer los elementos básicos de un buen prompt.
- Idealmente haber completado los cursos 01 y 02 de esta ruta.

## Duración y metodología

- **16 horas** en 8 módulos.
- Cada módulo combina explicación, demostración, práctica y entregable.
- Al menos 60% del tiempo debe dedicarse a trabajo práctico.
- Cada participante trabaja sobre un proceso real de su vida profesional.

## Proyecto transversal

Desde el primer módulo, cada alumno selecciona una tarea real que:

- se repita al menos dos veces al mes;
- consuma una hora o más;
- produzca un entregable reconocible;
- tenga una fuente de información clara;
- pueda revisarse con criterios objetivos.

Durante el curso, esa tarea evoluciona:

```text
Chat asistido
→ Artifact o entregable
→ Proyecto con conocimiento
→ Flujo conectado
→ Delegación supervisada
→ Sistema documentado
```

## Regla de alcance

Este curso enseña **Claude como plataforma para trabajo profesional**. La configuración técnica profunda de CLAUDE.md, hooks, Skills, subagentes, plugins y CI/CD pertenece a la Especialización Claude Code.
