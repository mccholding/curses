| Lógico | ¿El proceso y la salida tienen coherencia? |
| Evidenciable | ¿Las afirmaciones importantes tienen fuente? |
| Accionable | ¿El resultado permite avanzar? |
| Repetible | ¿Puede ejecutarse de nuevo con variables distintas? |
| Seguro | ¿Respeta datos, permisos, límites y aprobaciones? |
