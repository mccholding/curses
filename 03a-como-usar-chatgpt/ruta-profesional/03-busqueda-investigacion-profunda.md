# Módulo 3 · Búsqueda e investigación profunda

**Duración:** 2 horas.

## Objetivos

- Elegir entre conocimiento general, búsqueda e investigación profunda.
- Diseñar una pregunta investigable.
- Controlar fuentes, alcance y fecha de corte.
- Revisar planes, citas, desacuerdos e incertidumbre.

## 3.1 Tres niveles

| Nivel | Úsalo para | Riesgo principal |
|---|---|---|
| Chat | explicación, ideas y conceptos estables | desactualización o falta de evidencia |
| Búsqueda | hechos recientes o consultas rápidas | resultados incompletos o fuentes débiles |
| Investigación profunda | preguntas de varios pasos y síntesis de múltiples fuentes | alcance mal definido, costo y exceso de confianza |

## 3.2 La pregunta investigable

Una buena solicitud incluye:

```markdown
# Decisión que debe apoyar
[qué se decidirá con el informe]

# Pregunta
[qué debe investigarse]

# Alcance
[sector, país, periodo, población o producto]

# Fuentes
[oficiales, académicas, internas o dominios permitidos]

# Criterios de comparación
[costo, riesgo, capacidad, evidencia]

# Fecha de corte
[fecha exacta]

# Entregable
[resumen, tabla, informe, presentación]

# Incertidumbres
[qué debe declararse como no resuelto]
```

## 3.3 Control del plan

Antes de iniciar una investigación extensa:

1. revisar la interpretación de la pregunta;
2. confirmar alcance y exclusiones;
3. ajustar las fuentes autorizadas;
4. establecer criterios de calidad;
5. definir fecha de corte;
6. especificar qué afirmaciones necesitan citas;
7. decidir cuándo detenerse.

El usuario debe intervenir cuando el plan se desvíe o aparezcan fuentes no adecuadas.

## 3.4 Jerarquía de fuentes

Orden recomendado:

1. fuente primaria oficial;
2. investigación o dataset original;
3. organismo regulador o institución reconocida;
4. medio especializado con metodología visible;
5. fuentes secundarias para contexto;
6. publicaciones sociales solo como señal, no como confirmación.

## 3.5 Tabla de evidencia

| Afirmación | Fuente | Fecha | Tipo | Confianza | Observación |
|---|---|---|---|---|---|

Toda afirmación que pueda afectar una decisión debe poder rastrearse.

## 3.6 Protocolo de desacuerdo

Cuando dos fuentes difieren:

1. no ocultar el desacuerdo;
2. comparar fechas;
3. revisar metodología;
4. distinguir universos o definiciones;
5. explicar qué fuente parece más aplicable;
6. mantener la incertidumbre si no puede resolverse.

## 3.7 Investigación con archivos y apps

La investigación puede combinar:

- web pública;
- sitios específicos;
- archivos subidos;
- apps conectadas de lectura;
- información interna autorizada.

Antes de usar datos conectados, definir:

- qué repositorios puede consultar;
- qué información está fuera de alcance;
- qué fuentes son oficiales;
- si el resultado puede compartirse;
- qué datos deben anonimizarse.

## 3.8 Errores comunes

- preguntar “investiga todo sobre…”;
- no especificar periodo;
- mezclar hechos y recomendaciones;
- aceptar una cita sin abrir la fuente;
- tratar una ausencia de evidencia como evidencia de ausencia;
- comparar cifras con definiciones distintas;
- usar investigación profunda para una consulta rápida;
- omitir cambios ocurridos después de la fecha de corte.

## Práctica guiada

Investigar una decisión real con al menos:

- tres fuentes primarias o institucionales;
- una fuente que contradiga o matice;
- tabla de evidencia;
- fecha de corte;
- sección de vacíos;
- recomendación condicionada.

## Prueba adversarial

El instructor introduce:

- una fuente desactualizada;
- una cifra sin metodología;
- dos definiciones incompatibles;
- un resultado de búsqueda atractivo pero no oficial.

El estudiante debe detectarlos y corregir el informe.

## Entregable

Informe documentado con resumen ejecutivo, metodología, tabla de evidencia, desacuerdos, incertidumbres, fecha de corte y recomendación verificable.
